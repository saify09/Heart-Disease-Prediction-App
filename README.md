# CardioGuard Pro | Advanced Heart Disease Prediction

This is a professional heart disease prediction web application developed by Saifuddin Hanif.

## Tech Stack
- **Frontend**: HTML5, Vanilla CSS3 (Glassmorphism), JavaScript (ES6)
- **Backend**: Python, Flask
- **Machine Learning**: Scikit-Learn (K-Nearest Neighbors), Joblib
- **Deployment**: Docker, Hugging Face Spaces

## Features
- Premium, modern UI with smooth animations.
- Real-time diagnostic risk assessment.
- Mobile-responsive design.
- Accurate prediction based on clinical parameters.
